Mis Proyectos

Cipher

Este proyecto tiene como objetivo cifrar y descifrar frases utilizando el método de cifrado César. El último consiste en desplazar un número determinado (que selecciona el usuario) de carácteres del alfabeto, tomando como base la posición del carácter original del mensaje. Por ejemplo, si la letra original del mensaje es "A", y se elije un número de desplazamiento de 4, entonces la letra que va a reemplazar a todas las "A" del mensaje será la letra "E", y así se hará con cada carácter. A la plataforma se le dio un enfoque de ayuda a mujeres en situación de riesgo.

Data Lovers

En este proyecto trabajo con datos almacenados en una API que contiene datos del Banco Mundial de Desarrollo. Se muestran todos aquellos datos que se relacionan con el desarrollo económico de las mujeres, tales como los demográficos, los laborales y los educativos. Únicamente se trabaja con datos de cuatro países de América Latina: Chile, Perú, México y Brasil. Todos los datos se muestran dinámicamente de acuerdo a lo que el usuario decide observar. Además implementa la observación de data a través de gráficos de Chart.js.
